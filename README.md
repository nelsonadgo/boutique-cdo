Plataforma de comercio electrónico desarrollada a medida para la venta de productos de Grow Shop, Medicinal y Accesorios. El sistema cuenta con un catálogo dinámico, carrito de compras, panel de administración para gestión de inventario y un chatbot integrado.

## ✨ Características Principales

* **Catálogo Dinámico:** Filtrado de productos en tiempo real por categorías.
* **Carrito de Compras:** Gestión de productos seleccionados con cálculo de totales.
* **Panel de Administrador (Backend):** * Sistema de login/registro (Usuarios y Roles).
  * Creación y gestión de nuevos productos en la base de datos.
  * Visualización de clientes registrados.
* **Asistente Virtual:** Chatbot interactivo flotante para responder preguntas frecuentes de los usuarios.
* **Diseño Responsivo:** Interfaz adaptable a dispositivos móviles y escritorio.

## 🛠️ Tecnologías Utilizadas

**Frontend (Interfaz de Usuario)**
* HTML5 y CSS3 (Diseño a medida, sin plantillas).
* Vanilla JavaScript (Lógica del carrito, filtros, chatbot y consumo de API).
* Despliegue en la nube: **Netlify**.

**Backend (Lógica y Servidor)**
* Python 3 y Flask (Creación de la API REST).
* MySQL (Base de datos relacional).
* Despliegue en la nube: **PythonAnywhere**.
